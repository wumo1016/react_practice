## React 17

- 没有什么明显的新特性 和 React16 的使用是一致的

## 安装脚手架

- `npm install -g create-react-app`

## jsx 语法

### 基本使用

- 变量、表达式 使用 `{}` 包裹
- class 设置 使用 className 设置
  - `<p className="title">设置 css class</p>`
- style
  - 写一个变量对象
  - 内联写法
- 原生 html
  - 标签上加 dangerouslySetInnerHTML 属性
  - 将字符串放在一个对象的 `__html` 属性上
- 加载组件
  - 和原生标签一样使用
